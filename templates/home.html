Internship Experience

Internship at Oracle

1. SEALS - Super Elastic Lift and Shift PDB Upgrade Harness for Autonomous DB

Technologies Used: Python, Oracle Multitenant Architecture, SQL, Python Multithreading

Description: Developed during my internship at Oracle, SEALS is an automated PDB upgrade tool designed to streamline database upgrades in Oracle’s cloud environments. The tool uses Python multithreading to manage batch upgrades, maximizing throughput while minimizing resource contention. It supports multiple upgrade methods and optimizes for performance by configuring batch sizes based on resource availability. The harness includes extensive error handling, logging, and monitoring, making it a reliable solution for large-scale upgrades.

Skills Gained: Parallel processing, resource optimization, database management, cloud infrastructure.
